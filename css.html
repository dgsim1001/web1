CSS

CSS는 HTML 및 JavaScript 와 함께 World Wide Web 의 초석 기술입니다. CSS는 레이아웃, 색상 및 글꼴을 포함하여 프레젠테이션과 콘텐츠를 분리 할 수 ​​있도록 설계되었습니다. 이러한 분리는 컨텐츠 접근성을 향상시키고 , 프리젠 테이션 특성의 사양에서 더 많은 유연성과 제어를 제공 하며, 구조적 컨텐츠의 복잡성과 반복을 줄여주는 별도의 .css 파일에 관련 CSS를 지정하여 여러 웹 페이지 에서 서식을 공유 할 수 있도록 합니다. .css 파일을 캐시 할 수 있도록 설정하여 파일을 공유하는 페이지와 해당 형식 간의 페이지로드 속도를 향상시킵니다. 서식과 내용을 분리하면 화면, 인쇄물, 음성 (음성 기반 브라우저 또는 화면 판독기 사용 ) 및 점자 기반 과 같은 다양한 렌더링 방법에 대해 서로 다른 스타일로 동일한 마크 업 페이지를 표시 할 수 있습니다. 촉각 장치. CSS에는 모바일 장치 에서 콘텐츠에 액세스하는 경우 대체 형식에 대한 규칙도 있습니다. 계단식 이름 은 지정된 우선 순위 체계에서 가져와 둘 이상의 규칙이 특정 요소와 일치하는 경우 적용되는 스타일 규칙을 결정합니다. 이 계단식 우선 순위 체계는 예측 가능합니다 CSS 사양은 W3C ( World Wide Web Consortium )에서 관리합니다. 인터넷 미디어 유형 ( MIME 유형 ) text/css은 RFC 2318 (1998 년 3 월)에 의해 CSS와 함께 사용하도록 등록되었습니다. W3C는 CSS 문서에 대한 무료 CSS 유효성 검사 서비스 를 운영합니다. HTML 외에도 다른 마크 업 언어는 XHTML , 일반 XML , SVG 및 XUL을 포함한 CSS 사용을 지원합니다.
